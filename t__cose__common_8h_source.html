163 /* Private value. Intentionally not documented for Doxygen. This is
164  * the size allocated for the encoded protected header parameters. It
165  * needs to be big enough for encode_protected_parameters() to
166  * succeed. It currently sized for one parameter with an algorithm ID
167  * up to 32 bits long -- one byte for the wrapping map, one byte for
168  * the label, 5 bytes for the ID. If this is made accidentially too
169  * small, QCBOR will only return an error, and not overrun any
170  * buffers.
171  *
172  * 17 extra bytes are added, rounding it up to 24 total, in case some
173  * other protected header parameter is to be added and so the test
174  * using T_COSE_TEST_CRIT_PARAMETER_EXIST can work.
175  */
176 #define T_COSE_SIGN1_MAX_SIZE_PROTECTED_PARAMETERS (1+1+5+17)
